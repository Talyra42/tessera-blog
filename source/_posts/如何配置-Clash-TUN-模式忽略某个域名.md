## 引言

本人目前在校大学生，由于日常学业需求，需要使用 Claude 之类的需要 VPN 的工具，并且由于 VS Code 的代理问题，需要开启 TUN 模式才可以正常运行；但是，本人学校的教务系统是纯内网访问的，如果需要访问教务，就需要手动关闭 VPN，然后访问，访问完了重新启动。

实话说，我觉得这样很麻烦，所以就研究了一下如何跳过这个域名。

> 注意，这个方法仅仅测试于本人的院校，不同学校可能是不同的，仅供参考！

## 具体步骤

话不多说，直接开始。首先，本人用的是 Clash Verge Rev（如需下载，自行去 Github 搜索下载），下方的图片均为软件画面。

首先，我们来到最下面的设置页面，找到 Clash 设置的 DNS 覆写部分：

![](/images/screenshot_20260610_212914.png)

这里有一个小的设置图标，点击一下，然后往下滑，找到 Fake IP 过滤和 域名服务器策略，设置一下校园网的后缀，以及对应的 IP 即可：

![](/images/screenshot_20260610_213001.png)

然后刷新一下 DNS 缓存，我们的校园网就可以正常访问啦！
