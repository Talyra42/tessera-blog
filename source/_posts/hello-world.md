欢迎来到 **Tessera** —— 这是我用 [Hexo](https://hexo.io/) 搭建的个人博客的第一篇文章。

<!-- more -->

## 常用命令

新建文章：

``` bash
hexo new "文章标题"
```

本地预览（默认 http://localhost:4000）：

``` bash
pnpm server
```

生成静态文件到 `public/`：

``` bash
pnpm build
```

更多用法见 Hexo 官方文档：[Writing](https://hexo.io/docs/writing.html) · [Server](https://hexo.io/docs/server.html) · [Generating](https://hexo.io/docs/generating.html)。
